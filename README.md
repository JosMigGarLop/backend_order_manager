# Order Manager - Backend

**Order Manager** es un sistema para la gestión de pedidos de productos físicos. Permite crear, editar, listar y gestionar pedidos de forma sencilla mediante una API REST, pensada para ser consumida por un frontend.

Es un proyecto realizado como **prueba técnica de José Miguel García López**.

---

## Stack Tecnológico

- **Java 17 & Spring Boot:** Desarrollo del backend y creación de APIs REST robustas.
- **Spring Web:** Manejo de controladores y rutas REST.
- **Spring Data JPA:** Acceso y gestión de datos en PostgreSQL mediante ORM.
- **Spring Boot Validation:** Validaciones automáticas en DTOs y entidades.
- **PostgreSQL:** Base de datos relacional para almacenar pedidos y productos.
- **Lombok:** Reducción de código repetitivo en modelos y DTOs.
- **Springdoc OpenAPI (Swagger):** Documentación automática de la API.
- **Maven:** Gestión de dependencias y construcción del proyecto.
- **JUnit 5:** Pruebas unitarias e integración para garantizar la calidad del código.
- **Docker & Docker Compose:** Contenerización y despliegue fácil en diferentes entornos.
